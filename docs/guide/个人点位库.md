# 个人点位库

个人点位库是你私人的点位管理空间，支持完整的增删改查操作，支持分享点位，置顶点位。

## 新增点位

### 基本信息

1. 点击左侧
2. 点击右侧面板的 **"新增点位"** 按钮
3. 填写以下信息：

| 字段  | 说明                | 必填  |
| --- | ----------------- | --- |
| 标题  | 点位名称，如"B区窗户进攻瞬爆烟" | ✅   |
| 阵营  | 进攻（ATK）或防守（DEF）   | ✅   |
| 地图  | 选择对应地图            | ✅   |
| 英雄  | 选择使用的英雄           | ✅   |
| 技能  | 选择技能类型（C/Q/E/X）   | ✅   |

### 上传图片

支持上传以下类型的图片：

#### 站位图（必需）
- 标记投掷技能的站位位置
- 建议包含明显的参照物

#### 瞄点图（必需）
- 标记瞄准的参照物
- 建议使用准星对准的截图

#### 落点图（可选）
- 标记技能的落点位置
- 帮助验证技能效果

#### 站位图2 / 瞄点图2（可选）
- 用于需要多个站位或瞄点的复杂点位
- 点击"启用"按钮后显示

### 上传方式

::: tip 推荐：剪贴板上传
1. 在游戏中截图（通常是 F12）
2. 切换到浏览器
3. 点击"剪贴板上传"按钮
4. 图片自动上传到图床并显示预览
:::

**其他方式：**
- 点击"启用链接输入"，手动粘贴图片 URL

### 添加描述

为每张图片添加描述，说明：
- 站位的具体位置
- 瞄准的参照物
- 技能的使用时机
- 注意事项

### 视频来源（可选）

#### 剪贴板填入
1. 复制包含视频链接的分享文案
2. 点击"剪贴板填入"按钮
3. 系统自动提取链接

#### 手动输入
直接粘贴 B 站或抖音视频链接

#### 自动获取作者信息
- 输入链接后 1 秒自动获取
- 显示作者头像和昵称
- 支持手动刷新

详见：[作者信息获取](作者信息自动获取.md)

## 编辑点位

1. 在点位列表中找到要编辑的点位
2. 点击 **"编辑"** 按钮
3. 修改信息后点击 **"保存"**

::: warning 注意
编辑点位不会影响已分享到共享库的版本
:::

## 删除点位

### 单个删除
1. 点击点位卡片的 **"删除"** 按钮
2. 确认删除

### 批量清空
1. 点击左侧面板的 **"清空所有点位"** 按钮
2. 确认操作

::: danger 警告
删除操作不可恢复，请谨慎操作！
:::

## 筛选与查看

### 筛选条件

- **地图筛选**：选择特定地图
- **英雄筛选**：选择特定英雄
- **阵营筛选**：进攻 / 防守
- **技能筛选**：C / Q / E / X

### 查看详情

点击点位卡片查看：
- 所有图片（大图预览）
- 详细描述
- 作者信息（如果有）
- 视频来源链接（精准空降）

## 分享点位

1. 在点位详情中点击 **"分享"** 按钮
2. 系统生成短链接
3. 复制链接分享给他人

详见：[共享点位库](共享点位库.md)

## 最佳实践

### 命名规范
- ✅ 好的命名：`B区窗户进攻瞬爆烟`
- ❌ 不好的命名：`烟雾弹1`

### 图片质量
- 使用游戏内截图（F12）
- 确保参照物清晰可见
- 避免过度压缩

### 描述详细
- 说明具体站位位置
- 标注瞄准的参照物
- 补充使用时机和注意事项

### 视频来源
- 尽量添加视频来源
- 方便查看完整演示
- 给原作者引流

## 📌 置顶功能

将常用点位置顶，方便快速访问：

1. 在点位卡片上点击 **"置顶"** 按钮
2. 置顶的点位会显示在列表最前面
3. 再次点击可取消置顶

::: tip 跨设备同步
置顶数据会同步到云端，在其他设备登录同一账号后可看到相同的置顶设置。
:::

## 📥 批量下载

一键打包下载所有点位数据和图片：

### 使用方法

1. 在个人库页面点击 **"批量下载"** 按钮
2. 选择下载范围：
   - 当前筛选结果
   - 全部点位
3. 等待打包完成后自动下载

### 下载内容

下载的压缩包包含：
- `lineups.json` - 点位数据文件
- `images/` - 所有点位图片

### 使用场景

- ✅ 数据备份
- ✅ 迁移到其他账号
- ✅ 离线查看

::: warning 注意
批量下载功能需要图片来自可跨域访问的图床，否则部分图片可能无法下载。
:::

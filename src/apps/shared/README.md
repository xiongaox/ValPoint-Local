# shared

用途：用于存放共享库相关文件。

文件：
- main.tsx：共享库主入口。
- PendingSubmissionsTab.tsx：共享库待审投稿标签。
- ResetPasswordModal.tsx：共享库ResetPassword弹窗。
- SharedApp.tsx：共享库应用。
- SharedLoginPage.tsx：共享库登录页面。
- SharedMainView.tsx：共享库主视图。
- SharedRightPanel.tsx：共享库右侧面板。
- SubmitLineupModal.tsx：共享库投稿点位弹窗。
- SyncToSharedModal.tsx：同步To共享库弹窗。
- useSharedController.ts：共享库控制器。

子目录：
- components：共享库组件。
- logic：共享库逻辑。

# hooks

用途：用于存放Hook相关文件。

文件：
- useEscapeClose.ts：处理 ESC 键关闭弹窗。
- useIsMobile.ts：检测是否为移动端。
- useLineupActions.ts：点位操作逻辑。
- useLineupDownload.ts：点位图片下载与导出。
- useLineupFiltering.ts：点位筛选与搜索逻辑。
- useLineups.ts：获取与管理点位列表。
- useModalState.ts：全局弹窗状态管理。
- usePinnedLineups.ts：本地点位置顶。
- useUserProfile.ts：管理本地用户资料。
- useValorantData.ts：提供游戏元数据（地图、特工等）。
